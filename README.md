# TabZap (Chrome Extension)

A tiny one-click extension to instantly clear browser cache and hard-refresh the active tab.

## Brand
- Name: **TabZap**
- Type: Manifest V3 Chrome extension

## Install
1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select this folder: `/Users/ikra/Server/extensions/TabZap`

## Usage
- Open any tab
- Click **TabZap** icon
- It clears cache and reloads current tab using bypass cache

## Files
- `manifest.json`
- `background.js`
- `icons/16.png`, `32.png`, `48.png`, `128.png`
